# Go Backend Lite (Free)

Simple Go backend boilerplate dengan Gin, PostgreSQL, User Auth + JWT.

## Features
- Gin HTTP server
- PostgreSQL database
- User registration & login
- JWT authentication
- Protected route example

## Requirements
- Go 1.21+
- PostgreSQL 17+
- Postman (optional)
